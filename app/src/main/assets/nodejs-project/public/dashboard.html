Set up your first sensor

BLE Tripwire is the hub — the listening is done by small ESP32 boards you flash and place around the area you want covered. Nothing appears here until a sensor reports in.

  1. Get a board — an ESP32-S3 DevKitC (or any BLE-capable ESP32), about $8–15.
  2. Flash the firmware — sketches and a step-by-step guide are in the project repository.
  3. Set your hotspot name and password in the sketch, then power the node.
  4. Turn on your phone's hotspot — the node finds this app on its own.
